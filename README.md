# Pass Chick Celo
A Web3 project integrated with the Celo blockchain.
## Features
- Fast and cheap transactions on Celo
## Setup
1. Clone this repository for your Celo project.
## Backend Configuration
Run the Node.js backend to connect with Celo Minipay.
## Frontend
Next.js UI connecting to Celo wallets via AppKit.
## Smart Contracts
Foundry-based solidity contracts designed for Celo.
## Testing
Use `forge test` to test your Celo smart contracts locally.
## Deployment
Scripts provided to deploy on Celo Alfajores and Mainnet.
